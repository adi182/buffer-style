Cards

Cards can be atoms or organisms depending on their child elements. A card with a single <p></p> remains an atom while a card with multiple atoms is an organism.

Card

Use this atom as a base for any UI that requires a container for its content.

Example: 
<div class="card">
</div>

Card within Content → Queue that is empty

Use this modified atom to communicate an empty time slot with Content → Queue.

Example: 
<div class="card card--empty">
</div>

Card without a border

Use this modified card atom for popups and modals.

Example: 
<div class="card card--no-border">
</div>
