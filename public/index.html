Buffer Style Guide

Typography

We exclusively use Open Sans as the base typeface for the Buffer product.

Heading Level 1

Heading Level 1 is used for the primary heading of a page.

Example:

Heading Level 1

<h1>Heading Level 1</h1>

Heading Level 2

Heading Level 2 is used for the secondary headings of a page.

Example:

Heading Level 2

<h2>Heading Level 2</h2>

Heading Level 3

Heading Level 3 is used for the tertiary headings of a page.

Example:

Heading Level 3

<h3>Heading Level 3</h3>

Heading Level 4, 5, and 6

Heading Level 4, 5, and 6 are used for various headings of a page.

Example:

Heading Level 4

<h4>Heading Level 4</h4>
Example:
Heading Level 5
<h5>Heading Level 5</h5>
Example:
Heading Level 6
<h6>Heading Level 6</h6>

NavBars

NavBars are a template made up of different organisms and molecules that serve as navigational headers for the Buffer application.

NavBar

A brief description of the NavBar component.

Class Name

navbar

Description

This template is the primary navigational header for the Buffer application.

Contents

It's comprised of two molecules:

  • navbar-title
  • navbar-menu-left

and one organism:

  • navbar-menu-right
Class Name

navbar-title

Description

The navbar-title atom typically holds an image or text depicting the title of the page or product.

Contents
  • logo atom
Class Name

navbar-menu-left

Description

This molecule contains several atoms of which are list items.

The navbar-menu-left molecule may evolve to become an organism if it itself contains a molecule like the drop-down list in the navbar-menu-right molecule.

Class Name

navbar-menu-right

Description

This organism contains an atom and a molecule.

Class Name

navbar-menu-item

Description

navbar-menu-item remains an atom while holding a single item such as an image or a string of text. It evolves into a molecule if holding more multiple images or strings of text. It evolves into an organism it holds a molecule.

Rendered

Buffer
  • Queue
  • Library
  • Analytics
  • Help
  • Account

Profile Pictures

A blurb about profile pictures!

Profile Picture - Regular

Class Name

profile-picture

Description

An atom used with a social network icon.

CSS
.profile-picture {
  display: block;
  width: $globalSpacing * 4;
  height: $globalSpacing * 4;

  border-radius: $globalSpacing * 2;
}

Markup

<img class="profile-picture" src="#">

Rendered

Profile Picture with Icon

Class Name

profile-picture-with-icon

Description

A profile picture with a social network icon.

Contents

profile-picture-with-icon is a molecule made up of two atoms:

  • profile-picture
  • icon
CSS
.profile-picture-with-icon {
  position: relative;
}

.profile-picture-icon {
  position: absolute;
  top: $globalSpacing * 2;
  left: $globalSpacing * 2;

  width: $globalSpacing * 2; // 16px
  height: $globalSpacing * 2; // 16px

  border-radius: $globalSpacing;
}

Markup

<div class="profile-picture-with-icon">
  <img class="profile-picture" src="#">
  <i class="icon icon-twitter profile-picture-icon">
</div>

Rendered

Class Name
