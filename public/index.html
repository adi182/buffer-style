Introduction

This style guide is a work in progress. Our goal for the Buffer Style Guide is to provide a library of all atoms, molecules, organisms, templates, and pages that make up the Buffer web app and company website.

Our Style Guide was designed with these goals in mind:

  • To be the point of truth for our user interface for the Buffer web app.
  • To enable us to move faster with design that has been tested by the Growth team.
  • To introduce a level of data-driven thinking when building and designing.
  • To help make software development easier.
