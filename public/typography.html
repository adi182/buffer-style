Typography

We exclusively use Open Sans as the base typeface for the Buffer product.

Headings

Headings inherit all CSS properties and values from body. They keep their default weight, have shared CSS properties, and specific CSS properties.

Shared CSS properties applied to all headings give them a margin-top of 30px, a margin-bottom of 10px, and a color of $outer-space.

Heading 1

The h1 node has a specific font-size value of 36px and shares the subtitle class with the h2 node.

Example:

Sed id dui a massa mollis lacinia.

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Suspendisse sodales quam vitae ex commodo malesuada. Fusce luctus diam et mi luctus, quis pharetra mauris aliquet.

<h1>Sed id dui a massa mollis lacinia.</h1>
<p>Sed id dui a massa mollis lacinia. Integer a dolor interdum, elementum lacus vulputate, luctus ex. Praesent elit lorem, aliquet nec vestibulum vitae, feugiat eu urna.</p>

Paragraphs

Paragraphs inherit all CSS properties and values from body. Paragraphs have a single CSS property specific to the p node, which gives it a margin-bottom of 16px.

List Items

List Items inherit all CSS properties and values from body. They have two CSS properties specific to the li node, which gives it a margin-top of 8px and a margin-bottom of 8px.
