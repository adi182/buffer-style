Icons

An introduction to icons!

Icon

We use bi (an acronym for Buffer Icon) as a global namepsace for our icons. Each bi object is the base bi object. The only difference being how you call an individual icon.

Icon - Analytics

A little bit about the analytics icon.

Example: 
<i class="bi bi-analytics"></i>
