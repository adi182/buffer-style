Cards

Cards are a general-use object that hold different parts of the UI.

Card

This is the base object for the card object.

Example: 
<div class="card">
</div>

Card -- Empty

This is a modified card object. It doesn't introduce any new functionality nor UI. It simply modifies the card object's border-style.

Example: 
<div class="card card--empty">
</div>

Card -- No Border

This is a modified card object. It doesn't introduce any new functionality nor UI. It simply modifies the card object's border-style.

Example: 
<div class="card card--no-border">
</div>

Card - Post

The card-post object inherits from the base card object and adds further functionality and UI. This object is used when a post is displayed.
