Tags

Tags are atoms that communicate a specific state of information to the user. For example, the amount of posts queued up within a user's connected account.

Tag

Use this tag to communicate a subtle state of information.

Example:
Text Label 
<div class="tag">
  Text Label
</div>

Tag primary

Use this tag to communicate a primary state of information.

Example:
Text Label 
<div class="tag tag--primary">
  Text Label
</div>

Tag error

Use this tag to communicate an error state of information.

Example:
Text Label 
<div class="tag tag--error">
  Text Label
</div>
