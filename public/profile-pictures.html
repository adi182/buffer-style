Profile Pictures

Profile pictures can be atoms or molecules depending on their child elements. A profile picture remains an atom while a profile picture with a social network icon is a molecule. A profile picture communicates a familar connection to the user.

Profile Picture

Use this atom to communicate a user's connected profile.

Example: 
<img class="profile-picture" src="#">

Profile Picture when inactive

Use this atom to communicate a user's connected profile that is currently inactive.

Example: 
<img class="profile-picture profile-picture--inactive" src="#">

Profile Picture when used in MetaBar organisms

Use this atom when communicating an account in MetaBar organisms

Example: 
<img class="profile-picture profile-picture--small" src="#">

Profile Picture with Icon

Use this molecule to communicate a user's connected profile and the social network.

Example: 
<div class="profile-picture-with-icon">
<img class="profile-picture" src="#">
<div class="profile-picture-icon">
  <i class="bi bi-circle-twitter"></i>
</div>
</div>

Profile Picture with Icon when inactive

Use this molecule to communicate a user's connected profile that is currently inactive.

Example: 
<div class="profile-picture-with-icon profile-picture--inactive">
<img class="profile-picture" src="#">
<div class="profile-picture-icon">
  <i class="bi bi-circle-twitter"></i>
</div>
</div>
