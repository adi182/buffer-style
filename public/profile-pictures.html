Profile Pictures

Profile Pictures are objects designed to display a user's profile picture.

Profile Picture

This is the base object for the profile-picture object.

Example: 
<img class="profile-picture" src="#">

Profile Picture -- Inactive

This is a modifier for the profile-picture object, which grayscales our customers' profile picture. When used with the profile-picture-with-icon, it also grayscales the social network icon.

Example: 
<img class="profile-picture profile-picture--inactive" src="#">
Example: 
<div class="profile-picture-with-icon profile-picture--inactive">
  <img class="profile-picture" src="#">
  <div class="profile-picture-icon">
    <i class="bi bi-circle-twitter"></i>
  </div>
</div>

Profile Picture -- Small

This is a modified profile-picture object. It doesn't introduce any new functionality nor UI. It simply modifies the profile-picture object's width and height.

Example: 
<img class="profile-picture profile-picture--small" src="#">

Profile Picture with Icon

This is the base object for the profile-picture-with-icon object that includes a profile picture and a social network icon. It includes two separate objects (profile-picture and icon) that each inherit from their own base objects.

Example: 
<div class="profile-picture-with-icon">
  <img class="profile-picture" src="https://pbs.twimg.com/profile_images/694861974564409344/WpVYdDTj_bigger.jpg">
  <div class="profile-picture-icon">
    <i class="bi bi-circle-twitter"></i>
  </div>
</div>
