Profile Pictures

Profile pictures can be atoms or molecules depending on their child elements. A profile picture remains an atom while a profile picture with a social network icon is a molecule. A profile picture communicates a familar connection to the user.

Profile Picture

Use the ProfilePicture atom to show a user's connected profile.

Example: 
<img alt="A profile picture of Steve Dixon" class="ProfilePicture" src="#" />

A ProfilePicture atom using the --small modifier:

Example: 
<img alt="A profile picture of Steve Dixon" class="ProfilePicture ProfilePicture--small" src="#" />

A ProfilePicture atom using the --large modifier:

Example: 
<img alt="A profile picture of Steve Dixon" class="ProfilePicture ProfilePicture--large" src="#" />
