Buttons

Buttons can be atoms or organisms depending on their child elements. A button with a text label remains an atom while a button with a text label and an icon (or any other atom) is an organism. A button communicates an action that will happen once a user has clicked/tapped it.

Button

Use this atom to communicate a primary action to the user.

Example: 
<button class="button">Text Label</button>

Button secondary

Use this atom to communicate a secondary action to the user.

Example: 
<button class="button button--secondary">Text Label</button>

Button tertiary

Use this atom to communicate a tertiary action to the user.

Example: 
<button class="button button--tertiary">Text Label</button>

Button warning

Use this atom to communicate a destructive action to the user.

Example: 
<button class="button button--warning">Text Label</button>

Button inactive

Use this atom to communicate a currently locked action to the user.

Example: 









<button class="button button--inactive">Text Label</button>

<button class="button button--secondary button--inactive">Text Label</button>

<button class="button button--tertiary button--inactive">Text Label</button>

<button class="button button--warning button--inactive">Text Label</button>
