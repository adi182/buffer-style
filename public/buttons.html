Buttons

A button object executes code in response to user interactions. When you click or tap a button, the button performs any actions attached to it. You communicate the purpose of a button using text, an icon, or both. The appearance of buttons is configurable, so you can modify buttons for different use cases. You can add buttons to your interface using the markup below.

Button

This is the base object. Use this button when communicating a primary action to the user.

Example: 
<button class="button">Add to Queue</button>

Button -- Secondary

This is a modified base object. Use this modifier when communicating a secondary action to the user.

Example: 
<button class="button button--secondary">Add to Queue</button>

Button -- Tertiary

This is a modified base object. Use this modifier when communicating an alternative action to the user.

Example: 
<button class="button button--tertiary">Add to Queue</button>

Button -- Warning

This is a modified base object. Use this modifier when communicating a destructive action to the user.

Example: 
<button class="button button--warning">Delete Post</button>

Button with Icon

This is the base object with an icon object and text.

Example: 




<button class="button"><i class="bi bi-plus"></i>Add to Queue</button>
<button class="button button--warning"><i class="bi bi-x"></i>Remove from Queue</button>
<button class="button button--secondary"><i class="bi bi-plus"></i>Add to Queue</button>
