Typography

We exclusively use Open Sans as the base typeface for the Buffer product.

Heading Level 1

The Heading Level 1 object is used for the primary heading of a page. Semantically, a h1 might not always be styled below i.e. if using a h1 in a navbar, it will appear differently than in the example below. The example below shows an unaltered, base style for our h1 object.

Example:

Heading Level 1

<h1>Heading Level 1</h1>

Heading Level 2

The Heading Level 2 object is used for the secondary headings of a page.

Example:

Heading Level 2

<h2>Heading Level 2</h2>

Heading Level 3

The Heading Level 3 object is used for the tertiary headings of a page.

Example:

Heading Level 3

<h3>Heading Level 3</h3>

Heading Level 4, 5, and 6

The Heading Level 4, 5, and 6 objects are used for various headings of a page.

Example:

Heading Level 4

<h4>Heading Level 4</h4>
Example:
Heading Level 5
<h5>Heading Level 5</h5>
Example:
Heading Level 6
<h6>Heading Level 6</h6>
