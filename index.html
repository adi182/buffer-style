Buttons

We use buttons to initiate actions inside and outside of the Buffer app. less/components/Button.less provides a number of predefined button styles, which are documented below, for most use cases.

Button

This is the default style of a button. To define it, assign a button or a node with a class of Button.

Modifiers

Button--state-warning

Use this modifier to communicate a destructive action to the user.

Button--type-borderless

Use this modifier to communicate a non-primary action to the user.

Button--type-in-app

Use this modifier to communicate an in-app action to the user.

Button--type-secondary

Use this modifier to communicate a secondary action to the user.

Button--type-small

Use this modifier to communicate a primary action to the user when space is limited.

Button--type-tertiary

Use this modifier to communicate a tertiary action to the user.

BlockIcon

Use to communicate an action to the user using an icon only.
