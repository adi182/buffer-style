Grid System

Our Grid System is a 100%-width responsive 12-column grid that helps structure our product and marketing pages. The borders shown on our columns in this Style Guide are limited to this Style Guide. These borders are not a part of our GridSystem.scss. :)

GridRows

The GridRow object groups rows of columns.

Example: 
<div class="GridRow">
  Content
</div>

Columns

Columns are used to construct content within rows. Each collection of columns needs to add up to 12. Whether that's 12 single columns or three single columns, four single columns and six single columns grouped together on a single row.

GridContainer

A GridContainer allows us to create specific width wrappers that are centered to the display.

Example:
Twelve columns, using GridContainer 
<div class="GridRow">
  <div class="container">
    <div class="TwelveColumns--medium TwelveColumns--large">
      Twelve columns
    </div>
  </div>
</div>
