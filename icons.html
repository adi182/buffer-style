Icons

Icons are atoms that communicate in a visual language of what a user might expect from its use. For example, using alongside "Add to Queue" might indicate that the user will be "adding something".

Icon - Analytics

Use this icon to communicate analytics.

Example: 
<i class="bi bi-analytics"></i>

Icon - Arrow - Down

Use this icon to communicate a down action.

<i class="bi bi-arrow-down"></i>
