Notifications

Notifications are a general-use organism that hold information as a result of a particular action a user has taken or has to take.

Notification

Use this organism to communicate a standard notification to the user that requires no action.

Example:

This is a regular notification bar.

<div class="notification">
  <p>This is a regular notification bar.</p>
  <a href="#"><i class="bi bi-x"></i></a>
</div>

Notification displaying a warning

Use this organism to communicate a warning notification to the user and an action for them to take to resolve the warning.

Example:

This is a warning notification bar. Action to Take

<div class="notification notification--warning">
  <p>This is a warning notification bar. <a href="#">Action to Take</a></p>
  <a href="#"><i class="bi bi-x"></i></a>
</div>
