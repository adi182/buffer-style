Buttons

Buttons can be atoms or molecules depending on their child elements. A button with a text label remains an atom while a button with a text label and an icon (or any other atom) is a molecule. A button communicates an action that will happen once a user has clicked/tapped it.

Button

Use this atom to communicate a primary action to the user.

Example: 
<button class="button ia">Text Label</button>

Button secondary

Use this atom to communicate a secondary action to the user.

Example: 
<button class="button ia button--secondary">Text Label</button>

Button tertiary

Use this atom to communicate a tertiary action to the user.

Example: 
<button class="button ia button--tertiary">Text Label</button>

Button without a border

Use this atom to communicate a non-primary action to the user.

Example: 
<button class="button ia button--borderless">Text Label</button>

Button warning

Use this atom to communicate a destructive action to the user.

Example: 
<button class="button ia button--warning">Text Label</button>

Button small

Use this atom to communicate a primary action to the user when space is limited.

Example: 
<button class="button button--small">Text Label</button>

Button inactive

Use this atom to communicate a currently locked action to the user.

Example: 
<button class="button ia button--inactive">Text Label</button>
