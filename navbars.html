NavBars

NavBars are a template that display navigational organisms and typically stick to the top of a browser's viewport.

NavBar

Use this template for the Buffer web app's primary navigation.

Example:
Awesome plan
  • Content
  • Analytics
  • Schedule
  • Settings
  • Help
  • Account 
<div class="navbar">
  <div class="navbar-title">
    <i class="bi bi-buffer"></i><strong>Awesome</strong> plan
  </div>
  <ul class="navbar-menu-left">
    <li class="navbar-menu-item">
      <a class="navbar-menu-item--active" href="#"><i class="bi bi-content"></i>Content</a>
    </li>
    <li class="navbar-menu-item">
      <a href="#"><i class="bi bi-analytics"></i>Analytics</a>
    </li>
    <li class="navbar-menu-item">
      <a href="#"><i class="bi bi-calendar"></i>Schedule</a>
    </li>
    <li class="navbar-menu-item navbar-menu-item--last">
      <a href="#"><i class="bi bi-settings"></i>Settings</a>
    </li>
  </ul>
  <ul class="navbar-menu-right">
    <li class="navbar-menu-item">
      <a href="#"><i class="bi bi-exclamation"></i>Help</a>
    </li>
    <li class="navbar-menu-item navbar-menu-item--last">
      <a href="#"><img class="navbar-menu-item-profile-picture profile-picture" src="#">Account</a>
    </li>
  </ul>
</div>
