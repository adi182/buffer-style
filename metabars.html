MetaBars

MetaBars are organisms that display meta data to the user. They are always found as part of the Post templates.

MetaBar within Content → Queue

This MetaBar organism is used as part of the regular Post templates.

Example:
09:42 (GMT)
via Extension
  • Delete
  • Edit
  • Move to Top
  • Share Now
<div class="metabar">
  <div class="metabar-time">09:42 <span class="metabar-time-zone">(GMT)</span></div>
  <div class="metabar-via">via Extension</div>
  <ul class="metabar-menu">
    <li class="metabar-menu-item"><a href="#"><i class="bi bi-x"></i>Delete</a></li>
    <li class="metabar-menu-item"><a href="#"><i class="bi bi-edit"></i>Edit</a></li>
    <li class="metabar-menu-item"><a href="#"><i class="bi bi-arrow-up"></i>Move to Top</a></li>
    <li class="metabar-menu-item"><a href="#"><i class="bi bi-arrow-right"></i>Share Now</a></li>
  </ul>
</div>

MetaBar within Content → Posts

This modified MetaBar organism is used as part of a Post template with analytics.

  • 200 Clicks
