Colors

Each color Card has its Less variable and its hex value. Each Card contains current use cases for the color and in some cases, the opacity of that color.

Blues

Our blue color palette is used to signify a primary actionable item and its various states of interaction. The use of this color might be on a link that we want the user to focus on above others, or a button that we'd like to have more importance than another.
